Set a start and end hour to have that stop display only between those hours so that the page does not become too crowded.

For example, set the buses for your morning commute to display from 0-12, and buses for evening commute from 12-23.

If you do not want to show stops depending on hour just leave the fields blank.

If you want to show all stops, check the show all stops option.
